About:

The aim of this project is to solve a javascript kata and use a HTML and vanilla CSS frontend to show how the algorithm is solving the problem.

Press the Run » button, and any numbers which are touching will be combined. For example, [1, 2, 2, 2, 3] will become [1, 6, 3], because the 2s are next to each other.

The Kata:

"Implement a function sumConsecutiveDuplicates() which adds together all the consecutive numbers in an array and pushes them into a new array. For extra credit, write a function reduceConsecutives() which recursively uses your sumConsecutiveDuplicates() function to reduce the array down until there are no more consecutive duplicate numbers."

(Original kata from Northcoders wesbite)

Features:

The solution algorithm can be run once or recursively. The function will analyse each array element from left to right, and any duplicate elements are combined. However, this may result in more consecutive duplicates being created. The algorithm can be run recursively, and will continue to make analysis passes of the array until the resulting array cannot be reduced any further.

The javascript functions and html frontend are dynamic. The input array can be randomised using the drop down menu above, and the program will still function correctly.

This page is responsive, and the layout was designed with a "mobile first" approach.
